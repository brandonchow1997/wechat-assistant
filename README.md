# wechat-assistant
微信操控的电脑小助手。
# 功能
功能1：可以实现远程拍摄电脑前的用户，添加人脸识别组件可以识别当前用户。
功能2：可以运行cmd命令，远程关机/待机，或者打开指定应用程序。
